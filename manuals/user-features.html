주요 기능

대시보드

시스템의 핵심 정보와 현황을 한눈에 볼 수 있는 메인 화면입니다.

대시보드 구성 요소

  • 요약 카드: 중요한 통계와 지표를 카드 형태로 표시
  • 최근 활동: 사용자의 최근 활동 내역과 업데이트
  • 알림 센터: 시스템 알림과 중요 공지사항
  • 빠른 접근 메뉴: 자주 사용하는 기능에 빠르게 접근
  • 차트 및 그래프: 데이터의 시각적 표현

대시보드 활용 팁

  • 대시보드는 로그인 후 기본으로 표시되는 화면입니다
  • 각 위젯을 클릭하면 상세 정보를 확인할 수 있습니다
  • 화면을 새로고침하면 최신 데이터로 업데이트됩니다
  • 모바일에서는 세로로 위젯이 재배열됩니다

💡 대시보드 개인화

사용자 설정에서 대시보드에 표시할 위젯을 선택하고 순서를 변경할 수 있습니다.

프로필 관리

개인 정보와 계정 설정을 관리하는 방법을 안내합니다.

프로필 정보 수정

  1. 상단 메뉴에서 사용자 이름을 클릭하고 '프로필 설정'을 선택합니다
  2. 개인 정보 탭에서 이름, 이메일 등 기본 정보를 수정합니다
  3. 변경사항을 입력한 후 '저장' 버튼을 클릭합니다

비밀번호 변경

  1. 프로필 설정에서 '보안' 탭을 클릭합니다
  2. 현재 비밀번호를 입력합니다
  3. 새 비밀번호를 입력하고 확인합니다
  4. '비밀번호 변경' 버튼을 클릭합니다

알림 설정

받고 싶은 알림의 종류와 방법을 설정할 수 있습니다:

  • 이메일 알림: 중요한 업데이트를 이메일로 수신
  • 브라우저 푸시: 실시간 알림을 브라우저에서 표시
  • SMS 알림: 긴급한 상황에서 문자 메시지 수신

⚠️ 프로필 관리 주의사항

  • 이메일 주소는 계정 복구에 사용되므로 정확히 입력하세요
  • 비밀번호는 8자 이상, 영문/숫자/특수문자 조합을 권장합니다
  • 개인 정보 변경 시 관련 부서에 별도 통보가 필요할 수 있습니다
