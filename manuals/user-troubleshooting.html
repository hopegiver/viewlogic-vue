문제 해결

자주 발생하는 문제

사용자들이 자주 겪는 문제들과 해결 방법을 안내합니다.

로그인 관련 문제

Q: 로그인이 되지 않습니다

A: 다음 사항들을 확인해보세요:

  • 아이디와 비밀번호를 정확히 입력했는지 확인
  • Caps Lock이 켜져있지 않은지 확인
  • 브라우저의 쿠키와 캐시를 삭제 후 재시도
  • 다른 브라우저에서 시도해보기
Q: 비밀번호를 잊어버렸습니다

A: 로그인 화면에서 '비밀번호 찾기'를 클릭하고 이메일 주소를 입력하세요. 비밀번호 재설정 링크가 이메일로 발송됩니다.

화면 표시 문제

Q: 화면이 제대로 표시되지 않습니다

A: 브라우저 호환성 문제일 수 있습니다:

  • 최신 버전의 Chrome, Firefox, Safari, Edge 사용 권장
  • 브라우저 확대/축소 비율을 100%로 설정
  • 브라우저 캐시 삭제 후 새로고침
  • 애드 블록 등의 확장 프로그램 일시 비활성화
Q: 모바일에서 터치가 제대로 되지 않습니다

A: 화면을 새로고침하거나 브라우저를 재시작해보세요. 그래도 문제가 지속되면 앱을 삭제 후 재설치하시기 바랍니다.

데이터 관련 문제

Q: 저장한 데이터가 사라졌습니다

A: 데이터는 자동으로 백업됩니다. 시스템 관리자에게 문의하여 데이터 복구를 요청하세요.

Q: 파일 업로드가 되지 않습니다

A: 다음 사항을 확인해주세요:

  • 파일 크기가 제한을 초과하지 않았는지 확인 (최대 10MB)
  • 허용된 파일 형식인지 확인 (.jpg, .png, .pdf, .docx 등)
  • 네트워크 연결 상태 확인
  • 브라우저를 새로고침 후 재시도

성능 관련 문제

Q: 시스템이 너무 느려요

A: 다음 방법을 시도해보세요:

  • 불필요한 브라우저 탭 닫기
  • 다른 프로그램들 종료하여 메모리 확보
  • 네트워크 연결 상태 확인
  • 브라우저 캐시 및 임시 파일 삭제

고객 지원

위의 해결 방법으로도 문제가 해결되지 않는 경우 다음 방법으로 지원을 요청할 수 있습니다.

지원 요청 방법

📞 전화 지원
  • 고객센터: 1588-0000
  • 운영시간: 평일 09:00 ~ 18:00
  • 점심시간: 12:00 ~ 13:00 제외
✉️ 이메일 지원
  • 이메일: support@example.com
  • 응답시간: 24시간 이내
  • 첨부파일: 스크린샷 포함 권장
💬 온라인 채팅
  • 실시간 채팅: 화면 우하단 채팅 버튼
  • 운영시간: 평일 09:00 ~ 17:00
  • 평균 대기시간: 5분 이내
🎫 티켓 시스템
  • 접수: 시스템 내 '문의하기' 메뉴
  • 장점: 문제 해결 과정 추적 가능
  • 응답: 우선순위에 따라 처리

지원 요청 시 필요한 정보

빠른 문제 해결을 위해 다음 정보를 미리 준비해주세요:

  • 사용자 계정 정보: 아이디, 이름, 부서
  • 발생 시각: 문제가 발생한 정확한 시간
  • 브라우저 정보: 브라우저 종류 및 버전
  • 오류 메시지: 화면에 표시된 정확한 오류 내용
  • 재현 단계: 문제가 발생하기까지의 순서
  • 스크린샷: 문제 상황을 보여주는 화면 캡처

🔍 원격 지원

복잡한 문제의 경우 사용자의 동의 하에 원격 지원 도구를 사용하여 직접 문제를 해결할 수 있습니다. 원격 지원 시 개인정보나 민감한 데이터는 미리 정리해 주시기 바랍니다.
