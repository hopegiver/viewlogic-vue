데이터 관리

데이터 조회

시스템에 저장된 데이터를 효과적으로 조회하고 분석하는 방법을 안내합니다.

기본 조회 기능

  • 목록 보기: 데이터를 표 형태로 확인
  • 상세 보기: 개별 항목의 전체 정보 확인
  • 검색: 키워드로 원하는 데이터 찾기
  • 필터링: 조건에 맞는 데이터만 표시
  • 정렬: 원하는 기준으로 데이터 정렬

고급 조회 기능

  • 날짜 범위 검색: 특정 기간의 데이터 조회
  • 다중 조건 검색: 여러 조건을 조합하여 정밀 검색
  • 저장된 검색: 자주 사용하는 검색 조건을 저장하여 재사용
  • 실시간 업데이트: 데이터 변경 시 자동으로 화면 갱신

조회 화면 사용법

  1. 좌측 메뉴에서 조회할 데이터 유형을 선택합니다
  2. 상단 검색창에 키워드를 입력하거나 필터를 설정합니다
  3. 목록에서 항목을 클릭하면 상세 정보를 확인할 수 있습니다
  4. 페이지 하단의 페이지 번호로 더 많은 데이터를 탐색합니다

💡 조회 성능 향상 팁

  • 가능한 한 구체적인 검색 조건을 사용하세요
  • 불필요한 필터는 제거하여 조회 속도를 높이세요
  • 대량 데이터 조회 시에는 날짜 범위를 제한하세요

데이터 내보내기

시스템의 데이터를 다양한 형식으로 내보내어 외부에서 활용하는 방법입니다.

지원되는 내보내기 형식

  • Excel (.xlsx): 스프레드시트 편집에 최적화
  • CSV (.csv): 다른 시스템 간 데이터 교환에 유용
  • PDF (.pdf): 보고서나 문서 형태로 출력
  • JSON (.json): 개발자나 API 연동용

데이터 내보내기 절차

  1. 내보낼 데이터를 조회하고 필요한 필터를 적용합니다
  2. 목록 상단의 '내보내기' 버튼을 클릭합니다
  3. 원하는 파일 형식과 내보낼 항목을 선택합니다
  4. '다운로드' 버튼을 클릭하여 파일을 저장합니다

내보내기 옵션

  • 전체 데이터: 검색 결과의 모든 데이터
  • 현재 페이지: 화면에 표시된 데이터만
  • 선택된 항목: 체크박스로 선택한 데이터만
  • 커스텀 범위: 사용자가 지정한 범위의 데이터

⚠️ 데이터 내보내기 주의사항

  • 대용량 데이터 내보내기는 시간이 오래 걸릴 수 있습니다
  • 개인정보나 민감한 데이터는 별도 승인이 필요할 수 있습니다
  • 내보낸 파일은 보안을 위해 안전한 곳에 보관하세요
  • 일부 데이터는 사용자 권한에 따라 내보내기가 제한될 수 있습니다
