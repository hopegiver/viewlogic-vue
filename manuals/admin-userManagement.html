사용자 관리

사용자 추가

새로운 사용자 계정을 생성하는 방법입니다.

  1. 관리자 패널에서 '사용자 관리' 메뉴를 클릭합니다
  2. '새 사용자 추가' 버튼을 클릭합니다
  3. 필수 정보(이름, 이메일, 비밀번호)를 입력합니다
  4. 사용자 권한을 선택하고 '저장' 버튼을 클릭합니다

참고: 이메일 주소는 고유해야 하며, 비밀번호는 최소 8자 이상이어야 합니다.

사용자 정보 수정

기존 사용자의 정보를 수정하는 방법입니다.

  1. 사용자 목록에서 수정할 사용자를 선택합니다
  2. '편집' 버튼을 클릭하여 정보 수정 화면을 엽니다
  3. 필요한 정보를 수정하고 '저장' 버튼을 클릭합니다

주의: 사용자의 권한을 변경할 때는 신중하게 결정해야 합니다.

사용자 삭제

사용자 계정을 삭제하는 방법입니다.

  1. 사용자 목록에서 삭제할 사용자를 선택합니다
  2. '삭제' 버튼을 클릭합니다
  3. 삭제 확인 대화상자에서 '확인'을 클릭합니다

경고: 사용자를 삭제하면 해당 사용자의 모든 데이터가 영구적으로 삭제됩니다. 삭제 전에 반드시 백업을 확인하세요.
