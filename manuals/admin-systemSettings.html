시스템 설정

일반 설정

시스템의 기본적인 설정을 관리합니다.

  • 시스템 이름: 애플리케이션의 표시 이름을 설정합니다
  • 기본 언어: 시스템의 기본 언어를 한국어/영어 중 선택합니다
  • 시간대: 서버 시간대를 설정합니다 (기본: 서울/한국)
  • 이메일 설정: SMTP 서버 정보를 구성합니다
  • 파일 업로드: 최대 파일 크기 및 허용 확장자를 설정합니다

설정 변경 방법

  1. 시스템 설정 메뉴에서 '일반 설정' 탭을 클릭합니다
  2. 변경할 항목의 값을 수정합니다
  3. '저장' 버튼을 클릭하여 변경사항을 적용합니다
  4. 일부 설정은 시스템 재시작이 필요할 수 있습니다

보안 설정

시스템의 보안과 관련된 중요한 설정들을 관리합니다.

  • 비밀번호 정책: 최소 길이, 복잡성 요구사항 설정
  • 로그인 시도 제한: 연속 실패 시 계정 잠금 설정
  • 세션 타임아웃: 비활성 상태에서 자동 로그아웃 시간 설정
  • IP 접근 제한: 특정 IP 주소나 대역에서만 접근 허용
  • 2단계 인증: OTP나 SMS를 통한 추가 인증 활성화
  • SSL/TLS: HTTPS 연결 강제 및 인증서 관리

⚠️ 보안 설정 주의사항

  • 보안 설정 변경 시 기존 사용자에게 미리 공지해야 합니다
  • 너무 엄격한 정책은 사용자 편의성을 해칠 수 있습니다
  • IP 접근 제한 설정 시 관리자 본인이 차단되지 않도록 주의하세요
  • 모든 보안 설정 변경 사항은 로그에 기록됩니다
