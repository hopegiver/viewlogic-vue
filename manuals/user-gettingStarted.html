시작하기

로그인

시스템에 처음 접속하는 방법을 안내합니다.

  1. 웹 브라우저에서 시스템 주소에 접속합니다
  2. 관리자로부터 제공받은 아이디와 비밀번호를 입력합니다
  3. '로그인' 버튼을 클릭하여 시스템에 접속합니다

🔐 로그인 관련 주의사항

  • 비밀번호는 대소문자를 구분하니 정확히 입력하세요
  • 5회 이상 로그인에 실패하면 계정이 일시적으로 잠글 수 있습니다
  • 공용 컴퓨터에서는 사용 후 반드시 로그아웃하세요
  • 비밀번호는 정기적으로 변경하시기 바랍니다

로그인 문제 해결

  • 비밀번호를 잊어버렸을 때: 로그인 화면에서 '비밀번호 찾기'를 클릭하세요
  • 계정이 잠겼을 때: 시스템 관리자에게 문의하세요
  • 브라우저 오류: 쿠키와 캐시를 삭제한 후 다시 시도하세요

화면 구성 및 탐색

시스템의 기본적인 화면 구성과 탐색 방법을 설명합니다.

화면 구성

  • 상단 메뉴바: 주요 기능에 빠르게 접근할 수 있는 메뉴
  • 사이드 메뉴: 세부 기능들이 분류되어 있는 좌측 메뉴
  • 메인 콘텐츠 영역: 선택한 기능의 내용이 표시되는 중앙 영역
  • 사용자 정보: 로그인한 사용자 정보와 설정 메뉴

기본 탐색 방법

  • 메뉴 클릭: 상단이나 좌측 메뉴를 클릭하여 해당 기능으로 이동
  • 브레드크럼: 현재 위치를 확인하고 이전 단계로 돌아가기
  • 검색: 상단 검색창을 통해 원하는 내용 빠르게 찾기
  • 뒤로가기 버튼: 브라우저의 뒤로가기 버튼으로 이전 페이지 이동

모바일에서의 사용

모바일 기기에서는 다음과 같이 이용할 수 있습니다:

  • 좌측 메뉴는 햄버거 메뉴(☰) 버튼을 눌러 열 수 있습니다
  • 터치 제스처로 스크롤과 확대/축소가 가능합니다
  • 가로/세로 화면 모두 자동으로 최적화됩니다
